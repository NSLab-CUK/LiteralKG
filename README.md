# LiteralKG
LiteralKG is a novel Attributed Knowledge Graph Embedding Model developed by NS Lab, CUK based on pure [PyTorch](https://github.com/pytorch/pytorch) backend.

## Setup environment for running:

- Running: `python -m venv LiteralKG`

### Connect to the environment:

- On window OS run: `env.bat`
- On linux OS or MAC OS run: `source image/bin/activate`

### Install pip packages:

`pip install -r requirements.txt`

### To build model:

- Running `python index.py`
